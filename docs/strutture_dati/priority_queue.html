Coda di priorità

In una coda vengono estratti gli elementi uno alla volta in ordine di inserimento.
La coda di priorità permette di estrarre gli elementi uno alla volta in base ad una priorità: gli elementi con priorità maggiore vengono estratti prima.

L'implementazione della coda di priorità in C++ è data da std::priority_queue<TIPO>.

La priority queue è implementata con un max heap.

Priorità

Per poter utilizzare la priority_queue è necessario definire una priorità.

La priorità NON è intesa come un punteggio che viene attribuito ad ogni elemento: specifichiamo solo che un elemento ha più priorità di un altro, senza però attribuire un livello di priorità assoluto.

std::priority_queue utilizza l'operator < per stabilire l'ordine di estrazione.
Dati elementi di tipo TIPO possiamo specificarne la priorità così:

bool operator <(const TIPO& elemento1, const TIPO& elemento2) {
    if ( /* elemento1 più prioritario di elemento2 */ ) return false;
    else /* elemento1 MENO prioritario di elemento2 */ return true;
}

Nel caso non sia possibile ridefinire l'operator < (perché magari, in un certo contesto, dire che un elemento è minore di un altro ha un altro significato), è possibile fare:

bool compara(const TIPO& elemento1, const TIPO& elemento2) {
    // TODO: stessa implementazione dell'`operator <`...
}

// ...

// Purtroppo dobbiamo esplicitare il tipo del contenitore: `priority_queue` di default usa `vector`.
priority_queue<TIPO, vector<TIPO>, decltype(compare)&> q(compare);

// Nel caso sia sufficiente ordinare per priorità crescente (ps: va definito l'`operator >`)
priority_queue<TIPO, vector<TIPO>, greater<Tipo>> q;

Interfaccia

La priority_queue mette a disposizione un'interfaccia per inserire ed estrarre elementi:

  • push(TIPO& elemento): inserisce un nuovo elemento.
  • top(): prende il primo elemento in coda. (Attenzione che in queue si ha front()).
  • pop(): estrae il primo elemento in coda.

Costo

Inserimento e rimozione hanno costo logaritmico.
